# Mysterium Bounty transfers

Installation
------------
* **Step 1.** Install Node.js

```bash
sudo apt-get install nodejs
sudo apt-get install npm
```

* **Step 2.** Install dependencies

```bash
npm install
```

* **Step 3.** Run script. Woolia! Go for beer!

```bash
geth --fast --cache=512 --rpc
node balance.js
```
